Types of blocks - Markdown  


Render markdown directly in the page.

1. Select `Markdown` from the block menu

1. Insert markdown URL (with `.md` shortcut)

1. Publish documentation

> Be warned:  
> Markdown content is visible in the published documentation only

### Inserting markdown files from GitGub

1. Open markdown file in GitGub file browser, e.g. `README.md`

1. Click on *Raw* option from the file navigation bar (as seen on the image below).

1. Copy the URL from the browser. It should start with `https://raw.githubusercontent.com`

> Some extra info:  
> We support [GitHub Flavored Markdown format](https://github.github.com/gfm/#strikethrough-extension-) (e.g. tables or todo list)

### Known limitations

- We support only markdown files that are accessibly from a public URL. 

- We do not currently support `mdx` extension.

- Rendered markdown files are not indexed by search and Table of Contents.
